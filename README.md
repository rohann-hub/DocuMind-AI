An end-to-end AI chatbot system that answers user queries based on custom text data using Retrieval-Augmented Generation (RAG). The project utilizes HuggingFace embeddings to convert text into vector representations, FAISS for efficient similarity search, and Ollama as a local large language model for response generation. This solution enables fast, accurate, and fully offline question answering, making it suitable for document analysis, knowledge bases, and enterprise AI applications.
